gallery.js сверху есть объект с которым вы и должны работать.
Чуть ниже есть пример как вставлять данные в HTML.
То есть когда вы будете итерироваться по подготовленному массиву объекту то для каждого отдельного объекта в каждой итерации цикла
вы вставляете данные как показано в рабочем примере.
То есть отдельный объект item для примера, вставлять THML вы будете в цикле.
Так же для примера представлен простейший пример работы с темплейтами.

1) Копируете все себе в папку.
2) добавляем две константы в которых будет указваться до сколько элементов на скрине отображать (по умолчанию все) и четные/нечетные/все 
3) сначала работаем с объектом с помощью функций forEach, map, filter или других которые подходят для фильтрации начального объекта по заданным константам (параметрам)
получаем отфильтрованную версию объекта
4) далее снова работаем с только что полученным объектом и приводим данные в нужный вид:
- для имени: с помощью функций работы со стрингами делаете первую букву большой, остальные маленькие (ДЖИП -> Джип)
- для описания: с помощью функций работы со стрингами делаете обрезание до 15 символов. (пока не важно что оно обрежет половину слова, пока пусть так и будет)
-для даты: создать объект даты из заданных милисекунд и потом отобразить в виде "2015/07/02 14:15"

5) то что в итоге получилось, выводим в HTML используя для вдохновения пример который я оставил в коде. В процессе работы сам пример или переделываем или удаляем.
6) все это реализовываем с помощью функций 
7) в HTML коде убираем статическую часть чтоб осталась только динамическая
